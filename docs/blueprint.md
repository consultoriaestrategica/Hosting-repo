# **App Name**: Guardian Angel

## Core Features:

- User Roles: User authentication with roles: Admin, Staff, and Family.
- Patient Profiles: Capture Resident demographics, history, and family contact information.
- Daily Evolution Log: Record vital signs, medication, and general condition; view historical records.
- Emergency Notifications: Send alerts to family via email and WhatsApp using Twilio.
- Dashboard: Interactive dashboard for admins/staff showing key metrics like resident age, distribution of illnesses and dependency level.
- Patient Report Generation: Generates PDF reports of patients with their key info for different stakeholders.

## Style Guidelines:

- Primary color: Soft lavender (#D1C4E9) to evoke a sense of calm and care.
- Background color: Light gray (#F5F5F5), providing a clean and unobtrusive backdrop.
- Accent color: Pale blue (#A5D8EF), used sparingly to highlight interactive elements and important information.
- Body and headline font: 'Alegreya', a humanist serif that conveys elegance, intellect, and a contemporary feel.
- Use simple, line-based icons that are easy to understand, even for users who are not tech-savvy.
- Employ a clean, spacious layout with intuitive navigation.
